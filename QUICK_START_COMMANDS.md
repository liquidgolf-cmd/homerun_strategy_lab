# Quick Start Commands

## Navigate to Project First!

You need to be in the project directory before running backend/frontend commands.

### Step 1: Go to Project Directory

```bash
cd "/Users/michaelhill/Documents/GitHub/Homeruns Strategy Lab"
```

### Step 2: Start Backend (Terminal 1)

```bash
cd backend
npm run dev
```

### Step 3: Start Frontend (Terminal 2 - New Terminal Window)

```bash
cd "/Users/michaelhill/Documents/GitHub/Homeruns Strategy Lab"
cd frontend
npm run dev
```

Before running `npm run dev`, verify you're in the right directory:

```bash

```

Should show: `/Users/michaelhill/Documents/GitHub/Homeruns Strategy Lab/backend`

Or:

```bash

```

Should show files like: `package.json`, `src/`, `tsconfig.json`, `.env`
